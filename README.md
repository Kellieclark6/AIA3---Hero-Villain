GenAI Usage

- Which LLM(s) did you use?
  Claude code tutor - utilized in other project. Brizan Ok'd it, but asked that I not copy and paste all prompts due to 
  how it is used. The tutor is not allowed to code for me. Instead, it provides guidance to help me work through writing 
  the program using course content or valid sources like Python docs. It cannot guide me to code the way it would.

- What was your series of prompts?
  Discussions with tutor covered:
  - Understanding the code provided in logic.py and logic_test.py and how to use it. Then I applied it to the program
  - Understanding the Python concepts from Python docs and why they were needed
  - Translating the knowledge from class on proposition logic and knowledge bases to better understand how to implement
    things correctly. There was a lot of back and forth here because the theory did not always match the step by step code.

- Did you find this use helpful?
    I did find it helpful, however there were times it tried to take me down wrong, inefficient paths: like building a
    truth table before inputting the facts, which would make the program build unnecessary worlds and increase runtime.
    This is where I needed to push back and say I think there is a better way to do that. 
    
    It was very helpful in walking me through the various lines of code needed from logic.py and Python docs so I
    understood why we used it and what each line was doing.

- What did you learn from its use?
    This assignment more so than others had some growing pains. The tutor had a couple back and forth moments where it
    guided me to do something but then when we progressed to the next step, it realized the guidance it provided didn't
    work for program execution. This resulted in wasted time and an update to the guardrails.

    It reinforced that while you may not know everything, you need to have a firm understanding of what you are trying
    to accomplish and how the theory from class applies to the assignment. This is what allowed me to push back and have
    a dialogue about how to appropriately code the program as well as implement the understanding of how KBs and
    propositional logic work.
